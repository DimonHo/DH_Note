## 查询  
### 聚合查询  
#### 桶聚合(terms)  
`search_type=count` : 仅返回聚合结果  
`_term` : 按key排序  
`_count` : 按value排序  
```json
curl -XPOST http://127.0.0.1:9200/indexname/typename/_search?search_type=count
{
  "aggs": {
    "aggsResult": {
      "terms": {
        "field": "journalName",
        "size": 0,
        "order": { "_term": "asc" }
      },
      "aggs": {
        "aggsKeywords": {
          "terms": {
            "field": "keywords",
            "size": 10,
            "order": { "_count": "desc" }
          }
        }
      }
    }
  }
}
```

### 空值查询
#### exists Filter
```JSON
SELECT tags FROM posts WHERE tags IS NOT NULL;

GET /my_index/posts/_search
{
  "query": {
    "filtered": {
      "filter": {
        "exists": {
          "field": "tags"
        }
      }
    }
  }
}
```

#### missing Filter
```JSON
SELECT tags FROM posts WHERE tags IS NULL;

GET /my_index/posts/_search
{
  "query": {
    "filtered": {
      "filter": {
        "missing": {
          "field": "tags"
        }
      }
    }
  }
}
```
